Setup Takes 30 Seconds

Works in both Claude Code and Cowork. Pick whichever you use.

🤝 Cowork (Desktop App)

  1. Download the .skill file above
  2. Open Cowork, go to Customise, then Skills
  3. Click "+" and import the file
  4. Start a new session and say "write me a reel script" — it'll ask for your voice first

💻 Claude Code (Terminal)

  1. Copy the SKILL.md into your .claude/skills/reel-scriptwriter/ folder
  2. Open Claude Code in your project directory
  3. Say "write me a reel script"
  4. First run: paste 2-3 of your real reels so it learns your voice

Common Questions

What's new in v2? +

Everything that made v1 scripts feel generic is gone. v2 learns your voice from your real reels before writing, writes both demo scripts (30-55s) and story scripts (45-90s), uses hooks with real June 2026 view counts, refuses to write thin scripts without real substance, and tells you when to repost a winner.

Do I need a paid Claude plan? +

The skill works on any Claude plan. Cowork requires Pro or above. Claude Code requires Pro or Team. If you're on the free tier, the skill still works but you'll hit usage limits faster.

Does it only work for AI/tech content? +

Not at all. It's designed for any niche: fitness, cooking, finance, beauty, business, travel, parenting, and beyond. The voice profile is built from YOUR reels, so it adapts to whatever you make.

Can I use it for sponsored content? +

Yes. Just mention it's a sponsored video in your prompt. v2 opens with the result the product delivers, names the brand by second 5-8, and keeps disclosures in the caption — so it never sounds like an ad.

Does it write captions too? +

Yes — every script comes with a staccato-style caption, one keyword CTA, and 5-8 hashtags, ready to paste into Instagram, TikTok, or your scheduling tool.
