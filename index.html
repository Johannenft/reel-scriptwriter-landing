Setup Takes 30 Seconds

Works in both Claude Code and Cowork. Pick whichever you use.

🤝 Cowork (Desktop App)

  1. Download the SKILL.md file above
  2. Open Cowork, go to Customise, then Skills
  3. Click "+" and import the file
  4. Start a new session and say "write me a reel script"

💻 Claude Code (Terminal)

  1. Copy the SKILL.md into your .claude/skills/reel-scriptwriter/ folder
  2. Open Claude Code in your project directory
  3. Say "write me a reel script"
  4. Answer 2 prompts: your niche and your topic

Common Questions

Do I need a paid Claude plan?+

The skill works on any Claude plan. Cowork requires Pro or above. Claude Code requires Pro or Team. If you're on the free tier, the skill still works but you'll hit usage limits faster.

Does it only work for AI/tech content?+

Not at all. It's designed for any niche: fitness, cooking, finance, beauty, business, travel, parenting, and beyond. Just tell it your niche in the first prompt and it adapts everything.

Can I use it for sponsored content?+

Yes. Just mention it's a sponsored video in your prompt. The skill generates scripts that naturally integrate brand deals without losing authenticity or engagement.

What if I don't know my hook style?+

That's the whole point. You get 3 options with different hook types every time. Pick the one that feels right. Over time you'll learn which hooks your audience responds to best.

Does it write captions too?+

Yes. Every script includes a staccato-style caption and hashtag strategy. Copy and paste straight into Instagram, TikTok, or your scheduling tool.
